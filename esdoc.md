# ESDoc

## 官网

- ESDoc：https://esdoc.org/
- JSDoc：http://usejsdoc.org/

## 介绍

ESDoc 是一个根据 javascript 文件中注释信息，生成 JavaScript 应用程序或库、模块的 API 文档的工具。具有文档覆盖率统计、系统手册、一体化测试、详细接口说明等特点。

## ESDoc 与 JSDoc

 | ESDoc | JSDoc
:---: | --- | ---
ES标准 | ES6 以上 | ES6
模块化 | 只支持 ES6 | 支持 ES6、CommonJS、AMD
标签 | 只需要少量标签，自动语义化 | 标签完善，需要严格使用
文档 | 说明详细 | 需要自己定义
覆盖率 | 有 | 无
测试 | 有 | 无
手册 | 支持多个文档 | 支持多个文档
搜索 | 有 | 无
