# 使用插槽分发内容

目录



## 编译作用域

- 父组件模板的内容在父组件作用域内编译
- 子组件模板的内容在子组件作用域内编译

## 单个插槽

- 子组件不存在 < slot > 插口，父组件数据被丢弃
- 父组件的 innerHTML 会替换掉子组件 < slot > 的 outerHTML

```
<!-- 父组件 -->
<div>
  <h1>我是父组件的标题</h1>
  <my-component>
    <p>这是一些初始内容</p>
    <p>这是更多的初始内容</p>
  </my-component>
</div>

<!-- 子组件 -->
<div>
  <h2>我是子组件的标题</h2>
  <slot>
    只有在没有要分发的内容时才会显示。
  </slot>
</div>

<!-- 结果 -->
<div>
  <h1>我是父组件的标题</h1>
  <div>
    <h2>我是子组件的标题</h2>
    <p>这是一些初始内容</p>
    <p>这是更多的初始内容</p>
  </div>
</div>
```

## 具名插槽

- 存在 name 对应的，分别插入
- 不存在的合并一起当做单个插槽插入
- name 对应不上的，不插入

```
<!-- 父组件 -->
<app-layout>
  <h1 slot="header">这里可能是一个页面标题</h1>
  <p>主要内容的一个段落。</p>
  <p>另一个主要段落。</p>
  <p slot="main">这里有一些联系信息</p>
  <p slot="footer">这里有一些联系信息</p>
  <p>另两个主要段落。</p>
</app-layout>

<!-- 子组件 -->
<div class="container">
  <header>
    <slot name="header"></slot>
  </header>
  <main>
    <slot></slot>
  </main>
  <footer>
    <slot name="footer"></slot>
  </footer>
</div>

<!-- 结果 -->
<div class="container">
  <header>
    <h1>这里可能是一个页面标题</h1>
  </header>
  <main>
    <p>主要内容的一个段落。</p>
    <p>另一个主要段落。</p>
    <p>另两个主要段落。</p>
  </main>
  <footer>
    <p>这里有一些联系信息</p>
  </footer>
</div>
```

## 作用域插槽

```
<!-- 父组件 -->
<div class="parent">
  <app-layout>
    <template slot-scope="props">
      <span>hello from parent</span>
      <span>{{ props.text }}</span>
    </template>
  </app-layout>
</div>

<!-- 子组件 -->
<div class="app-layout">
  <p></p>
  <slot text="hello from child"></slot>
  <slot text="hello from child2"></slot>
  <div></div>
</div>

<!-- 结果 -->
<div class="parent">
  <div class="app-layout">
    <p></p>
    <span>hello from parent</span>
    <span>hello from child</span>
    <span>hello from parent</span>
    <span>hello from child2</span>
    <div></div>
  </div>
</div>
```

```
<!-- 父组件 -->
<div id="app">
    <div class="parent">
        <my-awesome-list :items="items">
            <!-- 作用域插槽也可以是具名的 -->
            <li
                    slot="item"
                    slot-scope="props"
                    class="my-fancy-item">
                {{ props.text }}
            </li>
        </my-awesome-list>
    </div>
</div>

new Vue({
    el: '#app',
    data: {
        items: [{text: 'd'},{text: 'j'},{text: 'h'}]
    },
});
    
<!-- 子组件 -->
Vue.component('my-awesome-list', {
    template: '<ul>\n' +
    '  <slot name="item"\n' +
    '    v-for="item in items"\n' +
    '    :text="item.text">\n' +
    '    <!-- 这里写入备用内容 -->\n' +
    '  </slot>\n' +
    '</ul>',
    props: ['items']
});

<!-- 结果 -->
<div id="app">
    <div class="parent">
        <ul>
            <li class="my-fancy-item">d</li>
            <li class="my-fancy-item">j</li>
            <li class="my-fancy-item">h</li>
        </ul>
    </div>
</div>
```
