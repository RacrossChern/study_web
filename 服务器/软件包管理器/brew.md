# brew 简单教程

## 安装 brew

    ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

## 安装软件

    brew install wget

## 卸载软件

    brew uninstall wget

## 使用 brew 查询软件

    brew search /wge*/

## 列出已安装的软件

    brew list

## 更新brew

    brew update

## 用浏览器打开brew的官方网站

    brew home

## 显示软件信息

    brew info

## 显示包依赖

    brew deps
